# Pricing

DeviceSDK is **free, open-source software** licensed under AGPL-3.0. There is no hosted service,
no subscription, and no per-message billing.

You run the server yourself on any hardware — Raspberry Pi, NUC, NAS, or any Docker host. You
own all the data. The only cost is the hardware you choose to run it on.

See the [README](https://github.com/device-sdk/devicesdk-monorepo) or the
[Quickstart](/docs/quickstart/) to get started in minutes.
