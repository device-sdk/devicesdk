# DeviceSDK Dashboard

Vue 3 + Quasar SPA for the DeviceSDK control panel.

## Development

From the monorepo root:

```bash
pnpm dev --filter @devicesdk/dashboard
```

## Lint

```bash
pnpm lint --filter @devicesdk/dashboard
```

## Build

```bash
pnpm build --filter @devicesdk/dashboard
```

## Type Check

```bash
pnpm check-types --filter @devicesdk/dashboard
```

## Configuration

See [Configuring quasar.config.js](https://v2.quasar.dev/quasar-cli-vite/quasar-config-js).
